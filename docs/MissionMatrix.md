# Mission Matrix v1

| Mission Type | Preset | Feel | Notes |
|---|---|---|---|
| Survive | Low | Forgiving | Good for onboarding |
| Survive | Standard | Balanced | Default experience |
| Survive | High | Stressful | Time pressure spikes early |
| Eliminate | Low | Controlled | Safe combat pacing |
| Eliminate | Standard | Aggressive | Risk/reward decisions |
| Eliminate | High | Chaotic | Constant escalation pressure |

## Interpretation

- Survive = time pressure (outlast the system)
- Eliminate = combat pressure (finish before collapse)
- Presets scale **when escalation hits**, not how combat works

## Usage

Use Mission Type + Preset combinations to test pacing:
- Tutorial: Survive + Low
- Default: Survive + Standard
- Challenge: Eliminate + High
